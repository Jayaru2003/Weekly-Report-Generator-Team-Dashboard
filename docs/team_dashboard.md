# Team Dashboard (Manager View)

The Team Dashboard is a dedicated, analytics-driven interface designed specifically for users with the `MANAGER` role. It provides a comprehensive overview of team operations.

## 1. Dashboard Summary
- Displays high-level Key Performance Indicators (KPIs).
- **Metrics**: Total reports submitted, overall compliance rate (percentage of team members who submitted on time), and pending reviews.

## 2. Activity Feed
- A chronological timeline of recent events within the team.
- **Events Tracked**: New report submissions, status changes (e.g., Report Approved/Rejected), new comments added, and project assignments.

## 3. Submission Status Tracker
- A quick-glance widget showing the current reporting status of all team members.
- Clearly highlights who has successfully submitted their report for the week and who is lagging or pending.

## 4. Trend Data Visualization
- Interactive charts (e.g., line or bar graphs) that display report submission trends over time.
- Helps managers identify patterns, such as periods of high productivity or consistent delays in reporting.

## 5. Workload by Project
- Visual breakdown of how the team's effort is distributed across active projects.
- Allows managers to see if certain projects are consuming disproportionate resources or if team members are over-allocated.

## 6. Advanced Filtering
- The dashboard includes robust filtering controls (e.g., comboboxes and date pickers).
- Managers can slice the data by specific **date ranges**, individual **team members**, or specific **projects**, instantly updating all graphs and summaries on the page.
